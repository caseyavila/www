Change font in Terraria 1.4

What you need

  • A copy of Terraria installed
  • The font you want to use installed
  • The official font extension
  • Microsoft Visual Studio C# 2010 Express (labeled as "Visual Studio 2010 VCS Express")
  • Microsoft Games for Windows Live
  • Microsoft XNA Game Studio 4.0

Instructions

  1. Install the Microsoft programs in the order listed above
  2. Launch Visual C# 2010 Express and select 'New Project'
  3. Under Visual C#, navigate to XNA Game Studio 4.0
  4. Select 'Windows Game 4.0'
  5. Save your project with Ctrl-S
  6. In the Solution Explorer on the right, right-click 'WindowsGame1Content (Content)'
  7. Under 'Add,' select 'New Item...'
  8. Select 'Spritefont' and add with the name as SpriteFont1.spritefont
  9. Extract font-extension.zip
  10. Right click ReLogic.Content.Pipeline.dll and select 'Properties'
  11. Check 'Unblock'
  12. Move ReLogic.Content.Pipeline.dll to the folder
    /Documents/Visual Studio 2010/Projects/WindowsGame1/WindowsGame1/WindowsGame1Content
  13. In Visual C#, right-click 'References' under 'WindowsGame1Content (Content)'
  14. Select 'Add Reference'
  15. Navigate to 'Browse' and add ReLogic.Content.Pipeline.dll as a reference
  16. Open SampleFont.dynamicfont with a text editor and copy all of the text
  17. Paste the text into SpriteFont1.spritefont within Visual C# (Delete the existing text before pasting)
  18. Save SpriteFont1.spritefont with Ctrl-S
  19. In the Properties window on the lower right, edit the 'Content Importer' and 'Content Processor' of SpriteFont1.spritefont to 'Dynamic Sprite Font Importer' and 'Dynamic Sprite Font Importer' respectively
  20. In the Solution Explorer, right-click 'WindowsGame1' and select 'Properties'
  21. Change the 'Game Profile' to 'Use Reach...'
  22. Replace 'Andy' with the name of the font you want to use (It must be installed on your computer)
  23. Change the font size to your preferred setting depending on experimentation and which font file you are choosing to replace See step 26 (You might not get the correct size on the first try)
  24. Press F6 to 'Build Solution'
  25. Navigate to Documents/Visual Studio 2010/Projects/WindowsGame1/WindowsGame1/WindowsGame1/bin/x86/Debug/Content/Fonts to locate the .xnb file
  26. Change the name of the .xnb file to either Death_Text.xnb, Item_Stack.xnb, Mouse_Text.xnb, or Title_Font.xnb, depending on which of the four font files you intend to replace (Each of these files require a different size to appear normal. Therefore, experimenting with sizes is necessary)
  27. Move the .xnb file to C:/Program Files (x86)/Steam/steamapps/common/Terraria/Content/Fonts (You may want to backup the original files)
  28. Launch Terraria and view your fonts, noting if they appear too small or large
  29. Repeat steps 22 - 28 for each of the 4 main font files, changing the size depending on your observations in step 28 (Combat_Text.xnb and Combat_Crit.xnb are generally considered to be fine without changing, however feel free to change them)
