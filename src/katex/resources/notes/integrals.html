Integrals

\(\displaystyle\int_{}^{}a^u\,du = \left(\dfrac{1}{\ln{a}}\right)a^u+C\)

\(\displaystyle\int_{}^{}\sin{u}\,du = -\cos{u}+C\)

\(\displaystyle\int_{}^{}\cos{u}\,du = \sin{u}+C\)

\(\displaystyle\int_{}^{}\tan{u}\,du = -\ln{\left|\cos{u}\right|}+C\)

\(\displaystyle\int_{}^{}\cot{u}\,du = \ln{\left|\sin{u}\right|}+C\)

\(\displaystyle\int_{}^{}\sec{u}\,du = \ln{\left|\sec{u}+\tan{u}\right|}+C\)

\(\displaystyle\int_{}^{}\csc{u}\,du = -\ln{\left|\csc{u}+\cot{u}\right|}+C\)

\(\displaystyle\int_{}^{}\sec^2{u}\,du = \tan{u}+C\)

\(\displaystyle\int_{}^{}\csc^2{u}\,du = -\cot{u}+C\)

\(\displaystyle\int_{}^{}\sec{u}\tan{u}\,du = \sec{u}+C\)

\(\displaystyle\int_{}^{}\csc{u}\cot{u}\,du = -\csc{u}+C\)

\(\displaystyle\int_{}^{}\dfrac{du}{\sqrt{a^2 - u^2}}\ = \arcsin{\dfrac{u}{a}}+C\)

\(\displaystyle\int_{}^{}\dfrac{du}{a^2 + u^2}\ = \dfrac{1}{a}\arctan{\dfrac{u}{a}}+C\)

\(\displaystyle\int_{}^{}\dfrac{du}{u\sqrt{u^2 - a^2}}\ = \dfrac{1}{a}\operatorname{arcsec}\dfrac{\left|u\right|}{a}+C\)
