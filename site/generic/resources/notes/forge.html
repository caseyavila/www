Install Forge without Java

What you need

  • A copy of Minecraft installed
  • The recommended release of Forge corresponding to your Minecraft version (listed as "Installer")

Instructions

  1. Move the downloaded .jar file to the location of your Minecraft java executable (This is often located in
    C:/Program Files (x86)/Minecraft Launcher/runtime/jre-x64/bin)
  2. Open a Powershell or CMD window
  3. In the window, navigate to the folder containing the .jar file using the cd command
  4. Run the command ./java -jar forge-1.8.9-installer.jar (change the name of the forge-1.8.9-installer.jar file in the aformentioned command to match the file-name of your own)
  5. Proceed through the Forge installation as normal
